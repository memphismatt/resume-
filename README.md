# Resume — Matthew Malone

This repository contains a single-page resume website intended for GitHub Pages.

Files:

- [index.html](index.html) — Landing page
- [styles.css](styles.css) — Styles and print rules
- [script.js](script.js) — Small interaction (copy LinkedIn)
- [resume.pdf](resume.pdf) — Placeholder PDF to be replaced with your real resume

Publishing on GitHub Pages

1. Push this repository to GitHub (if not already).
2. In the repo on GitHub, go to Settings → Pages.
3. Under "Source" select the `main` branch and the `/ (root)` folder.
4. Save. The site will be published at `https://<your-username>.github.io/<repo>` shortly.

Notes

- Replace `resume.pdf` with your final PDF (keep filename `resume.pdf`).
- All links are relative; the `Download PDF` button downloads `/resume.pdf`.
- The page is responsive and includes print-friendly rules.
